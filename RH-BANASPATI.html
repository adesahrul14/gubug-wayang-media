BANASPATI

Banaspati dalam bahasa kunonya di sebut khirtimuka. Ia dikenal sosok jin penjaga hutan belantara, dalam kepercayaan yang lain bansapati di kenal hantu dengan elemen api. Hnatu tersebut memiliki kekuatan yangh sangat besar, biasanya dipakai dukun untuk ilmu hitam. Banaspati dikenal mempunyai 2 wujud yang berbeda, yaitu berbentuk bola api yang hidup diantara pohon-pohon besar ditengah hutan atau berwujud manusia api. Kono wujud bansapati ketika menjadi bola api ukuranya sekepal tangan orang dewasa, namun apabila korbanya mengalami ketakutan maka bola api itu semakin membesar. Sementara dalam wujud manusia tubuhnya di selimuti api yang membara, ia berjalan dengan 2 tangan sementara kakinya menghadap ke atas, ia menjulurkan lidah api ke korbanya sehingga korbanya akan terbakar.
